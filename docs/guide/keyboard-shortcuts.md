# Keyboard Shortcuts

Speed up your editing workflow with these keyboard shortcuts.

## Global

| Shortcut | Action |
|---|---|
| `Ctrl/Cmd + N` | New Project |
| `Ctrl/Cmd + O` | Open Project |
| `Ctrl/Cmd + S` | Save Project |
| `Ctrl/Cmd + ,` | Open Settings |
| `Ctrl/Cmd + Q` | Quit |

## Video

| Shortcut | Action |
|---|---|
| `Space` | Play / Pause |
| `←` / `→` | Step frame backward / forward |
| `Shift + ←` / `→` | Jump 5 seconds backward / forward |
| `I` | Set In point |
| `O` | Set Out point |
| `Home` | Go to start |
| `End` | Go to end |

## Editing

| Shortcut | Action |
|---|---|
| `Ctrl/Cmd + Z` | Undo |
| `Ctrl/Cmd + Shift + Z` | Redo |
| `Ctrl/Cmd + X` | Cut |
| `Ctrl/Cmd + C` | Copy |
| `Ctrl/Cmd + V` | Paste |
| `Delete` | Delete selected |
| `Ctrl/Cmd + A` | Select all clips |

## Export

| Shortcut | Action |
|---|---|
| `Ctrl/Cmd + E` | Open Export dialog |
| `Ctrl/Cmd + Shift + E` | Quick export (last settings) |
